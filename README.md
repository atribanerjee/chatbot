## Setup instructions

```bash
# Enter Django project folder
cd backend
# Setup and activate virtualenv
virtualenv env
. env/bin/activate
# Install requirements
pip install -r requirements.txt
# Setup Django
./manage.py migrate
./manage.py runserver
# Then visit http://localhost:8000
```

## React js to javascript conversion instructions

```bash
# Enter Django project folder
cd frontend
# run webpack
npm run dev
```
